# Reverse-Proxy
Simple Reverse Proxy Using Python

```
usage: main.py [-h] [-pa PROXY_ADDRESS] [-pp PROXY_PORT]
               [-pda PROXY_DESTINATION_ADDRESS] [-pdp PROXY_DESTINATION_PORT]

optional arguments:
  -h, --help            show this help message and exit
  -pa PROXY_ADDRESS     proxy address
  -pp PROXY_PORT        proxy port
  -pda PROXY_DESTINATION_ADDRESS
                        proxy destination address
  -pdp PROXY_DESTINATION_PORT
                        proxy destination port
```
